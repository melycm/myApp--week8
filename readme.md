<h1>T.S Albums</h1>

<h2>Introduction</h2>
This is a back-end project which consist on a website where you can find all the information of Taylor Swift's albums, this features a feedback and a chat sections.


<h2>Technologies Used</h2>
<li>Node.js</li>
<li>Express</li>
<li>EJS</li>
<li>Body Parser</li>
<li>Socket.io</li>
